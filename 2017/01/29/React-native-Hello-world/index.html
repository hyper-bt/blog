React-native - Hello world

Jan 29, 2017

Step1

Using Homebrew to install Node.js.

Step2

Watchman is monitor tool from Facebook, We can use Watchman to monitor the change from code.

Step3

nam is Node Package Manager, it’s like Maven. It’s can help you to download the depend package for you project.

Step4

#projectName is your project name. react-native init will create a new React Native project. We can use #projectName.xcodeproj to open from Xcode.

Step5

Click to #projectName/ios/#projectName.xcodeproj (If you have permission problem, you can use chmod -R to change to permission)

Click Building... (Don’t forget to select your device)
